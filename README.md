# 🌍 AQI AI Smart Dashboard

A modern AI-powered dashboard that predicts Air Quality Index (AQI) using Machine Learning and provides real-time air quality insights.

---

## 🚀 Features

* 📊 Interactive AQI visualization (Plotly)
* 🤖 Machine Learning prediction (Random Forest)
* 📁 Upload custom dataset
* 🌐 Real-time AQI API integration
* ⚡ Fast model training (on-demand)
* 🎨 Modern 3D UI with glassmorphism
* 🏙 City-based prediction system

## 🧠 How It Works

1. Load dataset (default or uploaded)
2. Clean & preprocess data
3. Train model using **Train Model** button
4. Input pollution values
5. AI predicts AQI & category
6. Dashboard displays insights
